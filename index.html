Courses

COSC 4330: Fundamentals of Operating Systems:

Since network programming with sockets in C is a classic programming assignment for the OS course, the following resources may be helpful:

  • Beej's Guide to Network Programming Using Internet Sockets is a good site with lots of practical examples.
  • W. Richard Stevens' UNIX Network Programming may be relatively old, but it is comprehensive. If you want to learn just about everything about Unix sockets, this is it.
