Courses

COSC 4330: Fundamentals of Operating Systems:

Getting started with Unix

If you've never used the command line under Unix/Linux systems, you might want to check out for these guides

  • The Norman Matloff's Unix and Linux Tutorial Center has many short introductions to every tool you need to be proficient on a Unix system.
  • There are many guides at The Linux Documentation Project. You might want to check out "Introduction to Linux - A Hands on Guide".
  • You can check out the slides from Zaki Mughal's Unix workshop or the Intro to Linux workshop given at CACDS.
  • Books: "Linux for Dummies" and "Linux in a Nutshell" are pretty good and available at the UH Library.

IPC using Pipes and Shared Memory/Semaphores

  • Beej's Guide to Unix Interprocess Communication has examples for both using pipes and semaphores.
  • The Little Book of Semaphores covers the different ways you can use semaphores to solve different synchronisation problems with semaphores.

Network programming

Since network programming with sockets in C is a classic programming assignment for the OS course, the following resources may be helpful:

  • Beej's Guide to Network Programming Using Internet Sockets is a good site with lots of practical examples.
  • W. Richard Stevens' UNIX Network Programming may be relatively old, but it is comprehensive. If you want to learn just about everything about Unix sockets, this is it.
